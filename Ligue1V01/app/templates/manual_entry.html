Renseigne les informations du match, les joueurs, arbitres, cartons, buts, remplacements et observations. Le formulaire insère ensuite les données directement dans la base SQLite.

Format rapide des listes : une ligne par élément. Utilise le séparateur ;. Les champs non connus peuvent rester vides.

1. Informations du match

2. Joueurs domicile

Format : numéro; nom; licence; rôle; capitaine; gardien; poste; nationalité; notes

Staff domicile

Format : nom; rôle

3. Joueurs extérieur

Format : numéro; nom; licence; rôle; capitaine; gardien; poste; nationalité; notes

Staff extérieur

Format : nom; rôle

4. Officiels du match

Format : rôle; nom

5. Buts

Format : minute; équipe; joueur; détail

6. Cartons

Format : minute; équipe; joueur; couleur; motif. Couleur : jaune ou rouge.

7. Remplacements

Format : minute; équipe; joueur entrant; joueur sortant; détail

8. Observations et notes

Format : minute; auteur; niveau; note. Niveau : note, warning, critical.
